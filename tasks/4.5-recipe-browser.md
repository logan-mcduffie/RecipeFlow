**Features**:
- Search input with debouncing (300ms)
- Filter by recipe type (crafting, smelting, GT machine types)
- Results show item icon + name + recipe count
- Click to expand and see recipe variants
- Drag from result to canvas to add node
- Or click "Add to flowchart" button

**Recipe preview**:
- Expand item to see all recipe options
- Show inputs/outputs in compact form
- Click specific recipe to add that variant
