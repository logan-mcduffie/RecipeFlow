# Task 1.1: Initialize Monorepo Structure

**Phase**: 1 - Project Setup & Infrastructure
**Status**: Not Started

## Summary

Set up the project repository with workspaces for web app and companion mod.

## Why This Matters

Clean separation between the web application and Minecraft mod while sharing common types/schemas.

## Inputs/Outputs

- **Input**: Project requirements
- **Output**: Configured monorepo with build tooling

## Technical Notes

- Use pnpm workspaces or npm workspaces
- Structure:
  ```
  /packages
    /web          # React frontend
    /api          # Node.js backend
    /shared       # Shared types/schemas
  /mod            # Forge mod (separate build system)
  ```
- Configure TypeScript, ESLint, Prettier
- Add Docker Compose for local PostgreSQL

## Files to Touch

- `package.json` (root)
- `packages/*/package.json`
- `tsconfig.json`
- `docker-compose.yml`
- `.eslintrc`, `.prettierrc`

## Acceptance Criteria

- [ ] Monorepo structure created with workspaces
- [ ] TypeScript configured for all packages
- [ ] Docker Compose runs PostgreSQL locally
- [ ] Basic build/dev scripts work
