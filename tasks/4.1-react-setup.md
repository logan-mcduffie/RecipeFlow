# Task 4.1: React App Setup

**Phase**: 4 - Flowchart Editor Frontend
**Status**: Not Started

## Summary

Initialize React application with routing and base layout.

## Why This Matters

Foundation for all frontend functionality.

## Inputs/Outputs

- **Input**: Requirements
- **Output**: Running React app with navigation

## Technical Notes

- **Build tool**: Vite
- **Routing**: React Router
- **Styling**: TailwindCSS

**Routes**:
- `/` - Home/modpack browser
- `/m/:modpack` - Modpack page
- `/m/:modpack/flowcharts` - Community flowcharts
- `/m/:modpack/editor` - Flowchart editor
- `/m/:modpack/editor/:id` - Edit existing flowchart
- `/login` - OAuth login
- `/callback` - OAuth callback

**Base layout**:
- Header with logo, nav, user menu
- Sidebar (context-dependent)
- Main content area
- Responsive design

## Files to Touch

- `packages/web/src/App.tsx`
- `packages/web/src/routes/*`
- `packages/web/src/components/Layout.tsx`
- `packages/web/vite.config.ts`
- `packages/web/tailwind.config.js`
