**Modpack card info**:
- Icon/logo
- Name
- Version count
- Flowchart count
- Quick link to latest version

**Features**:
- Search by name
- Sort: alphabetical, popular, recent
- Featured/promoted section
- "Request modpack" for unsupported packs

**Request modpack flow**:
- Form to submit modpack request
- Link to CurseForge/Modrinth
- Goes to admin queue or GitHub issue
