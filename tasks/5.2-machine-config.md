# Task 5.2: Machine Configuration Panel

**Phase**: 5 - GregTech Calculations
**Status**: Not Started

## Summary

Panel to configure machine tier, parallels, and view calculations.

## Why This Matters

Users need to adjust parameters to model their actual setup.

## Inputs/Outputs

- **Input**: Selected recipe node
- **Output**: Configuration UI + calculated values

## Technical Notes

**Panel layout** (appears when node selected):
```
┌─────────────────────────────┐
│ Machine Configuration       │
├─────────────────────────────┤
│ Voltage Tier: [MV ▼]        │
│ Parallels:    [4      ]     │
│ Time Unit:    [/min ▼]      │
├─────────────────────────────┤
│ Calculations:               │
│ ─────────────────────────── │
│ Base EU/t:    32 EU/t       │
│ Effective:    128 EU/t      │
│ Duration:     2.5s          │
│ Power:        6,400 EU      │
│ ─────────────────────────── │
│ Throughput:                 │
│   Copper Wire: 96/min       │
│   Rubber:      48 L/min     │
└─────────────────────────────┘
```

**Controls**:
- Voltage tier dropdown: ULV, LV, MV, HV, EV, IV, LuV, ZPM, UV, UHV, UEV, UIV, UXV, OpV, MAX
- Parallel count input (validate: positive integer)
- Time unit selector: /sec, /min, /hour

**Displays**:
- Base EU/t (from recipe)
- Effective EU/t (after overclock)
- Effective duration
- Total power per operation
- Throughput per time unit (each output)

**Persistence**:
- Config saved per-node in flowchart data
- Survives save/load
