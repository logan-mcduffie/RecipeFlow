# Task 6.3: Export Features

**Phase**: 6 - Flowchart Persistence & Sharing
**Status**: Not Started

## Summary

Export flowcharts as image or JSON for external sharing.

## Why This Matters

Users want to share flowcharts outside the app (Discord, Reddit, forums, etc.).

## Inputs/Outputs

- **Input**: Current flowchart
- **Output**: PNG/SVG image or JSON file

## Technical Notes

**Export dialog**:
```
┌─────────────────────────────────┐
│ Export Flowchart                │
├─────────────────────────────────┤
│ Format:                         │
│ ○ PNG Image                     │
│ ○ SVG Vector                    │
│ ○ JSON Data                     │
│                                 │
│ Options:                        │
│ ☑ Include background            │
│ ☑ Include throughput labels     │
│ ☐ Transparent background        │
│                                 │
│ [Cancel]           [Export]     │
└─────────────────────────────────┘
```

**Image export**:
- Use React Flow's built-in export or html-to-image library
- PNG for raster (Discord-friendly)
- SVG for vector (scalable)
- Options: include/exclude background, transparency
- High resolution option for large flowcharts

**JSON export**:
- Full flowchart data structure
- Can be imported back into the app
- Useful for backup or transfer between instances

**Import**:
- Drag-and-drop JSON file onto editor
- Or "Import" button in menu
- Validate structure before importing

**Share link**:
- Copy shareable URL to clipboard
- Public flowcharts get permanent URLs
- Private flowcharts could have share tokens
