**Getting Started guide**:
1. Create account (OAuth login)
2. Find your modpack
3. Install companion mod
4. Run `/recipeflow sync` in-game
5. Create your first flowchart

**Companion mod guide**:
- Download links for each MC version
- Installation instructions
- Configuration (server URL, auth token)
- Running the sync command
- Troubleshooting common issues

**Editor guide**:
- Interface overview
- Creating/editing flowcharts
- Using auto-generation
- Configuring machine settings
- Keyboard shortcuts reference

**Self-hosting guide**:
- System requirements
- Docker installation
- Environment configuration
- Domain/SSL setup
- Backup procedures

**In-app help**:
- Tooltips on key UI elements
- "?" help button in toolbar
- Link to full docs
