# Task 8.1: Recipe Display Toggles

**Phase**: 8 - Polish & Deployment
**Status**: Not Started

## Summary

Options to show/hide intermediate tools, alternate recipes.

## Why This Matters

Reduces clutter when viewing complex chains. GregTech recipes often involve many intermediate tools (hammers, files, wire cutters) that can make flowcharts messy.

## Inputs/Outputs

- **Input**: Toggle settings
- **Output**: Filtered flowchart view

## Technical Notes

**Toggle options**:
```
┌─────────────────────────────────┐
│ Display Options                 │
├─────────────────────────────────┤
│ ☑ Show GT tools in crafting     │
│   (hammers, files, etc.)       │
│                                 │
│ ☑ Show alternate recipes        │
│   (when multiple paths exist)  │
│                                 │
│ ☐ Collapse repeated patterns    │
│   (group identical sub-chains) │
│                                 │
│ ☑ Show fluid connections        │
│                                 │
│ View mode: [Detailed ▼]        │
│   • Detailed (all info)        │
│   • Compact (minimal)          │
│   • Icons only                 │
└─────────────────────────────────┘
```

**Implementation**:
- GT tools toggle: filter out tool items from recipe displays (they're not consumed)
- Alternate recipes: show/hide secondary recipe paths
- Collapse patterns: detect repeated sub-graphs, show as single node with multiplier
- Applies to both existing flowcharts and auto-generation

**Persistence**:
- Save as user preferences (global)
- Or per-flowchart settings
